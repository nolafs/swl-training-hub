## [0.1.0] - 2026-01-22

### Added

- Initial project setup with Next.js 16 and TypeScript
- Tailwind CSS configuration
- Prismic CMS integration
- ModuleSlider component with horizontal scrolling
- ModuleCard component with hover animations and expanding color bar
- ModuleDetail modal with zoom animation, progress indicator, and navigation
- LessonSlider component for lesson navigation within modules
- LessonCard component with hover effects
- LessonDetail modal for lesson preview
- Route structure: homepage, module pages, lesson pages
- CSS Modules for component-scoped styling
- Mock data for development and testing

### Technical

- App Router architecture
- Client components for interactive UI elements
- TypeScript strict mode enabled
- ESLint configuration
